## Nix Shells

`nix-shell`はカスタムシェル環境を起動するコマンド。
そのカスタムシェルに好きなパッケージを利用できます。

Nix Shellファイルを`shell.nix`または`default.nix`にファイル名変更をすると自動的に読み込まれます。

* [Haskell](nix-shells/haskell.nix)
* [Rust](nix-shells/rust.nix)
