## NixOSとは

NixOSはNixパッケージマネジャーを利用ひたリナックスディストリビューション。

- 宣言型設定: 一つの設定ファイルでシステムをカスタマイズできます。
- 信頼できる: アトミックアップグレードとバージョンロールバックはできます。
- デプロイ向け: NixOpsツールでNixOSをVirtualBoxやAmazon EC2に展開できます。
